# 🌿 Temiz Dünya - Profesyonel Sürüm

Bu proje, gönüllü çevre temizliği için geliştirilmiş çift platformlu (Web & Mobil) profesyonel bir ekosistemdir.

## 🚀 Hızlı Başlatma (Tek Komut)

Tüm sistemi (Web + Mobil) aynı anda başlatmak için ana dizinde şu komutu çalıştırın:

```powershell
.\start_all.ps1
```

---

## 📱 Uygulama Özellikleri (Profesyonel Sürüm)

### 1. Web Dashboard (React + Vite)
- **Profesyonel Arayüz:** SaaS standartlarında, modern ve dinamik tasarım.
- **Canlı Aktivite Akışı:** Platformdaki temizlik hareketlerini anlık takip edin.
- **Gelişmiş İstatistikler:** Temizlenen alanlar ve gönüllü sayıları için görsel kartlar.
- **Tam Entegrasyon:** Harita, Gruplar ve Sosyal kanıt sistemi bir arada.

### 2. Mobil Uygulama (Flutter)
- **Canlı Harita:** Firestore tabanlı, gerçek zamanlı kirli alan ve görev takibi.
- **Premium SOS Sistemi:** 5 saniyelik geri sayımlı ve kategorize edilmiş acil yardım.
- **Görsel Kalite:** Yanıp sönen efektler, yumuşak geçişler ve modern "frosted glass" tasarımı.
- **Web Senkronizasyonu:** Web dashboard ile birebir aynı veri ve tasarım dili.

---

## 🛠️ Teknik Altyapı

- **Backend:** Firebase Firestore (Real-time data)
- **Web:** React 18, Vite, Custom Theme System
- **Mobil:** Flutter (Windows/Android/iOS), OSM Maps
- **Animasyonlar:** Flutter Animate & CSS Keyframes

---

## 📂 Dosya Yapısı

- `/flutter`: Flutter kaynak kodları.
