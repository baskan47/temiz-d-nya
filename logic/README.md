# purdunya_logic

Small Dart package that extracts core state logic from a Flutter widget and provides unit tests.

Quick start:

1. Install Dart SDK (see https://dart.dev/get-dart).
2. From this folder run:

```bash
dart pub get
dart test
```

What it contains:
- `lib/purdunya_logic.dart`: A pure-Dart `PurdunyaState` class with `joinGroup`, `startCleaning`, `finishCleaning`.
- `test/purdunya_logic_test.dart`: Unit tests using `package:test`.
